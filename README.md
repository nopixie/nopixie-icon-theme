## Installation

1. Open **Extensions** sidebar in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for `No Pixie Icon Theme`
3. Click **Install**
4. Click **Reload** to refresh your editor

Or install via command line:
```bash
code --install-extension no-pixie.nopixie-icon-theme
```

## Activation

To activate the icon theme:

1. Open Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Type `Preferences: File Icon Theme`
3. Select one of the available icon themes:
   - **No Pixie Purple Icons** - Classic purple variant
   - **No Pixie Blue Icons** - Cool blue/cyan variant
   - **No Pixie Yellow Icons** - Warm yellow/amber variant

## Icon Features

The icon themes include:

### Folder Icons
- Custom icons for common directories:
  - `src`, `dist`, `build`, `out`
  - `node_modules`, `.git`, `.github`, `.vscode`
  - `tests`, `test`, `__tests__`
  - `docs`, `documentation`
  - `config`, `configuration`
